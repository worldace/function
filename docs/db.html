dbクラス

データベースのSQLiteにアクセスするクラス。PDOを使います。
割り切った仕様のため、主キーの名前をidに限定しています。

new

dbオブジェクトを作成する

引数説明
1 dbファイルのパス
2 テーブル名
戻 dbオブジェクト
  • ファイルが存在しない時は、自動的に作成されます

table

現在のテーブルを変更する。

引数説明
1 テーブル名

table_create

テーブルを作成する。

引数説明
1 作成するテーブルの情報。連想配列で[列名=>型 制約]という形式

SQLiteで設定できる型

型説明
text文字列
integer整数
real小数点
numeric 数字
none何も変換しない

SQLiteで設定できる制約

制約説明
primary key主キーにする
autoincrement 自動採番する
not null値がない場合は、エラーになる
default 値値がない場合は、指定した値になる
unique値が重複した場合は、エラーになる
check(式)値が式の条件を満たさない場合は、エラーになる

table_keys

現在のテーブルの列名一覧を取得する

引数説明
- 引数はありません
戻 配列

insert

レコードを1件追加する

引数説明
1 追加するデータ。連想配列で[列名=>値]という形式
戻 追加されたid番号

update

レコードを1件更新する

引数説明
1 更新するid番号
2 更新するデータ。連想配列で[列名=>値]という形式

delete

レコードを1件削除する

引数説明
1 削除するid番号

select

データを取得する。引数により3パターンあります。

複数行取得では、idの大きい順から選択されます。逆順にしたい場合は第3引数にtrueを渡してください

search

文字列検索をする。

引数説明
1 検索ワード。空白で区切るとOR検索
2 検索対象の列名。列が複数ある場合は配列に入れる
3 開始位置。0から。idの大きい順に検索されます
4 取得件数
戻 配列＋連想配列の2重配列

query

任意のSQL文を実行する。プレースホルダにも対応

引数説明
1 SQL文
2 プレースホルダに対応する配列
戻 PDOStatementオブジェクト

count

現在のテーブルの全レコード数を取得する。

引数説明
- 引数はありません
戻 全レコード数

transaction

トランザクション処理を実行する。

引数説明
1 コールバック関数。トランザクションが必要な処理を記述する
失敗したら関数内で例外を投げてください。ロールバックされます
2 コールバック関数に渡したい引数。続けて第3引数以降にも渡せます。
戻 「コールバック関数の戻り値」がそのまま返ります

コールバック関数にはdbオブジェクトが渡されます。
コールバック関数に追加の引数を渡したい場合は、transaction()の第二引数以降に渡してください。

参考情報：PHP callable

データをオブジェクトで取得する

当クラスでの取得データは連想配列となりますが、指定クラスのオブジェクトにすることもできます。

テーブル名を指定する所を、クラス名にすると実現できます。
テーブル名と同名のクラスが存在し、クラス名をフルパスで渡すのが条件です。

テーブルに関するデータと操作を1つのクラスにまとめることができます。
