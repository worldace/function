response::json()

JSONを出力して、PHPを終了する

引数説明
1 JSONに変換するデータ。配列や連想配列など
2 アクセスを許可するオリジン(プロトコル＋ホスト名＋ポート番号)。複数ある場合には配列に入れる。
オリジンはhttp://www.example.comのように指定する。最後のスラッシュは不要。
省略時は全オリジンを許可します
戻 なし

上記例ではブラウザからCookieが送られてきません。Cookieを送信するようにするには、 JavaScript側では送信前にxhr.withCredentials = true;を追加し、PHP側では第2引数のオリジンを指定します
