response::cookie()

クッキーを出力する

引数説明
1 クッキーの名前
2 クッキーの値
省略時や空文字時はクッキーを削除します
3 設定があれば連想配列で(後述)
戻 なし

クッキーの設定

キー 値
expire クッキーの有効期限。省略時は50日後
path クッキーを有効にするパス。省略時はカレントディレクトリ
domain クッキーを有効にするドメイン。省略時は同ドメイン
secure https時のみクッキーを出力する。省略時はfalse
httponly JavaScriptからクッキーにアクセスできなくする。省略時はtrue

クッキーの設定方法はPHPのsetcookie()と同等です
