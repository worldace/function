js::api()

JavaScriptに適したAPIを提供する。
JavaScriptからPHPのメソッドを実行する感覚のAPIが作れます

引数説明
1 オブジェクト
このオブジェクトのメソッドがAPIとして提供されます
2 JSONの設定があれば連想配列で
response::jsonの第二引数と同じ内容
戻 なし

PHP側

事前準備としてクラスを作成します。このクラスのメソッドがAPIとなります。

メソッドの引数は外部からの入力となるため、セキュリティには注意を払う必要があります

プライベートメソッドとマジックメソッドはAPIから実行できません

JavaScript側

事前準備として api.js モジュールが必要です。
またモジュールファイル内で、PHP側へのURLを設定する必要があります。

api.js は IE に対応していません

JavaScript側ではPHPのメソッドを実行する感覚でOK。戻り値は常にPromiseです

ファイル送信

JavaScriptのFile/Blobオブジェクトに対応しているので、ファイルはデータとして渡せます
