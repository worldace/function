request::file()

アップロードされたファイルの情報を取得する。PHPの $_FILES 相当

引数説明
1 フォームのname名
戻 アップロードの情報が入ってる連想配列(後述)

この関数はアップロードの本処理は行いません。本処理は request::upload() をご利用ください

アップロード情報の連想配列
キー値
name元のファイル名
tmp_name一時保存の場所
sizeファイルサイズ(バイト)
typeファイルのMIMEタイプ
errorエラーコード

複数アップロード時

1つのnameで複数のファイルをアップロードする場合、戻り値は配列となり、各要素が結果の連想配列となります
